⚙️ 代理配置说明

AI 识别需要通过 Cloudflare Worker 代理访问 Anthropic API（解决浏览器跨域限制），配置一次永久有效，完全免费。

第一步：获取 Anthropic API Key
  1. 访问 console.anthropic.com
  2. 注册/登录 → API Keys → Create Key
  3. 复制 Key（以 sk- 开头）
第二步：部署 Cloudflare Worker
  1. 访问 dash.cloudflare.com，注册免费账号
  2. 左侧菜单 → Workers & Pages → Create
  3. 点 "Create Worker" → 部署
  4. 点 "Edit code"，粘贴下载的 cf-worker.js 内容
  5. 点右上角 "Deploy"
  6. 回到 Worker 页面 → Settings → Variables → Add variable
    变量名：ANTHROPIC_API_KEY
    变量值：你的 API Key（选 Encrypt）
  7. 保存后复制 Worker URL（如 https://xxx.workers.dev）
第三步：填入代理地址

将 Worker URL 粘贴到上方「代理地址」输入框，即可使用 AI 识别功能。
